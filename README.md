# 3D Force Graph Visualization

A simple project to visualize graph/network data in an interactive **3D force-directed layout** using Computer Vision and JavaScript.

---

## Features

- Interactive 3D graph visualization  
- Real-time node and link rendering  
- Smooth zoom, pan, and rotation  
- Dynamic force simulation  

---

## Topics & Tech Stack

- Computer Vision
- JavaScript  
- HTML/CSS  
- [3d-force-graph](https://github.com/vasturiano/3d-force-graph)
